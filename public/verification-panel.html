📋 Instructions

How to use this development tool

🚀 Quick Start

  1. Go to Registration Page
  2. Create a new account with your email
  3. Come back here and click the verification link
  4. Verify your email and login!

💡 Features

  1. One-Click Copy: Click the copy button next to any link
  2. Auto-Refresh: Links update automatically
  3. Expiration Status: See if links are still valid
  4. User Names: See who registered
  5. Clean Interface: Much better than console scrolling!

🔧 For Production

  1. Set up Gmail SMTP in .env.local
  2. Users will receive real verification emails instead
  3. See GMAIL_SETUP.md for instructions

🔐 Security

  1. Email Required: Users must verify email before login
  2. 24-Hour Expiry: Links expire for security
  3. Secure Tokens: 32-character random tokens
  4. One-Time Use: Tokens are cleared after verification
