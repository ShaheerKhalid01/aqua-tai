📋 Instructions

How to use this development tool

🚀 Quick Start

  1. Go to Forgot Password Page
  2. Enter your email and click "Send Reset Link"
  3. Come back here and click the link below
  4. Reset your password and login!

💡 Features

  1. One-Click Copy: Click the copy button next to any link
  2. Auto-Refresh: Links update automatically
  3. Expiration Status: See if links are still valid
  4. Clean Interface: Much better than console scrolling!

🔧 For Production

  1. Set up Gmail SMTP in .env.local
  2. Users will receive real emails instead
  3. See GMAIL_SETUP.md for instructions
